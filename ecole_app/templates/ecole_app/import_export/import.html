Format du fichier CSV

Le fichier CSV doit contenir une ligne d'en-tête avec les noms des champs, suivie des données.

Champs requis par type

  • nom - Obligatoire
  • prenom - Obligatoire
  • date_naissance - Format YYYY-MM-DD
  • telephone
  • email
  • adresse
  • classe - ID ou nom de la classe
  • creneau - ID ou nom du créneau

  • nom - Obligatoire

  • professeur - ID ou nom du professeur

  • jour - 1 (Lundi) à 7 (Dimanche)
  • heure - Format HH:MM

  • eleve - ID de l'élève ou "Nom Prénom" - Obligatoire
  • montant - Nombre décimal - Obligatoire
  • date - Format YYYY-MM-DD - Obligatoire
  • methode - ESP, CHQ, VIR ou CB
  • commentaire
Exemple
nom,prenom,classe,creneau
Dupont,Marie,CE1,Mercredi matin
Martin,Jean,CE2,Samedi après-midi
